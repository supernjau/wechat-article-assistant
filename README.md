# 公众号文章助手

一款用于微信公众号官方编辑器的 Chrome 扩展，帮助公众号运营者更高效地完成文章排版。

它目前提供三项功能：

- **HTML 导入**：将排版好的 HTML 文件或 HTML 代码写入公众号正文。
- **全文关键词强调**：批量强调文章中的指定关键词。
- **往期推荐**：读取近期发布的公众号文章，选择模板后插入“往期推荐”模块。

文章正文、HTML 文件和插件设置只在本地浏览器中处理，不会上传到外部服务器。

## 安装方法

本扩展暂未发布到 Chrome 应用商店，需要手动加载一次。以后正常使用时，无需重复安装。

### 第一步：下载插件

1. 打开本仓库页面。
2. 点击右上角绿色的 **Code** 按钮。
3. 点击 **Download ZIP**。
4. 下载完成后，解压 ZIP 文件。

也可以使用 Git 下载：

```bash
git clone https://github.com/supernjau/wechat-article-assistant.git
```

### 第二步：加载到 Chrome

1. 在 Chrome 地址栏输入 `chrome://extensions`，按回车。
2. 打开页面右上角的 **开发者模式**。
3. 点击左上角的 **加载已解压的扩展程序**。
4. 选择刚才解压得到的 `wechat-article-assistant` 文件夹。
5. 确认扩展列表中出现“公众号文章助手”。

### 第三步：在公众号后台启用

1. 登录[微信公众号平台](https://mp.weixin.qq.com/)。
2. 进入公众号文章编辑页面。
3. 刷新一次页面。
4. 页面右侧出现竖排的“文章助手”按钮，即表示加载成功。

点击 Chrome 工具栏中的扩展图标，再点击“打开文章助手”，也可以打开操作面板。

## 功能一：HTML 导入

适用于已经使用其他工具完成排版，希望将文章整体写入公众号正文的场景。

### 使用步骤

1. 打开公众号文章编辑页面。
2. 点击页面右侧的“文章助手”。
3. 选择 **HTML 导入**。
4. 点击 **选择 HTML 文件**，导入 `.html` 或 `.htm` 文件。也可以直接将 HTML 代码粘贴到输入框。
5. 在下方预览区检查正文效果。
6. 选择导入方式：
   - **覆盖正文**：清空当前正文，再写入导入内容。
   - **插入光标位置**：保留当前正文，并在光标所在位置插入内容。
7. 点击 **导入正文**。

### 注意事项

- 使用“覆盖正文”前，请确认当前编辑器中没有需要保留的内容。
- 扩展会保留正文结构、常见内联样式和图片。
- 为了安全，扩展会移除脚本、外部样式表、事件属性和危险链接。
- 导入后仍建议在公众号编辑器中完整预览一次文章。

## 功能二：全文关键词强调

适用于英语教学、知识科普和职场写作等内容。输入关键词后，可以一次性强调全文中的相同词语。

### 使用步骤

1. 点击页面右侧的“文章助手”。
2. 选择 **关键词强调**。
3. 输入需要强调的关键词。每行输入一个，也可以用中文或英文逗号分隔。
4. 选择强调样式：
   - 黄色标记
   - 红色加粗
   - 蓝色下划线
5. 点击 **应用强调**。

如果需要撤销本插件添加的强调，点击 **移除本插件强调**。这个操作只会移除插件添加的样式，不会影响你手工完成的排版。

## 功能三：往期推荐

“往期推荐”可以读取公众号后台近期发布的文章标题和链接，自动生成一个适合放在正文结尾的推荐模块。

这个功能需要配合公众号官方的“超链接”窗口使用。请严格按照下面的顺序操作。

### 第一步：打开公众号官方“超链接”窗口

1. 打开公众号文章编辑页面。
2. 在编辑器顶部工具栏中找到 **超链接**。
3. 点击 **超链接**，打开官方的“编辑超链接”窗口。
4. 在“链接内容”一栏中选择 **选择账号文章**。
5. 确认窗口中已经出现近期发布的文章列表。

此时不要关闭官方窗口，也不需要勾选某一篇文章。

### 第二步：读取近期文章

1. 保持官方“编辑超链接”窗口打开。
2. 点击页面右侧的“文章助手”。
3. 选择 **往期推荐**。
4. 点击 **读取最近文章**。
5. 插件会读取窗口中最靠前的近期文章，最多读取五篇。

读取成功后，请检查每一篇文章的标题和链接。

### 第三步：检查文章链接

公众号后台不同版本的页面结构可能存在差异：

- 如果插件成功读取链接，链接输入框会正常显示公众号文章地址。
- 如果官方窗口只显示标题，没有向页面暴露链接，插件仍会读取文章标题，但链接输入框会显示为黄色。

黄色输入框表示链接尚未补全。请手动粘贴对应的公众号文章链接，格式通常类似：

```text
https://mp.weixin.qq.com/s/...
```

为了避免生成无法点击的推荐项目，只要还有文章缺少有效链接，插件就不会执行插入操作。

### 第四步：调整推荐内容

读取完成后，可以继续调整：

- 修改文章标题。
- 修改或补充文章链接。
- 使用上移、下移按钮调整顺序。
- 删除不希望推荐的文章。
- 点击“添加文章”手动补充推荐内容。

模块最多保留五篇文章。

### 第五步：选择模板并插入

1. 从六种模板中选择一种：
   - 清爽列表
   - 数字目录
   - 编辑精选
   - 极简边框
   - 杂志索引
   - 工作室风格
2. 在“模板预览”中检查效果。
3. 选择插入位置：
   - **正文末尾**：适合常规的往期推荐。
   - **光标位置**：适合将模块插入指定段落之间。
4. 点击 **插入往期推荐**。
5. 回到公众号正文，确认推荐模块的位置和链接。

### 常见问题

#### 点击“读取最近文章”后没有读取到内容

请依次确认：

1. 是否已经打开公众号官方的“编辑超链接”窗口。
2. “链接内容”是否选择了“选择账号文章”。
3. 官方窗口中是否已经显示近期文章列表。
4. 是否在官方窗口仍然打开时点击了“读取最近文章”。
5. 是否已经刷新公众号文章编辑页面，使扩展加载最新版本。

如果仍然无法读取，可以点击“添加文章”，手动输入标题和公众号文章链接。

#### 为什么链接输入框是黄色的？

这表示插件读取到了文章标题，但公众号后台没有向页面暴露对应链接。补充链接后，黄色提示会消失。

#### 为什么无法插入往期推荐？

请检查是否还有黄色链接输入框。每一篇推荐文章都必须填写有效的公众号文章链接。

#### 为什么读取到的文章少于五篇？

插件最多读取五篇文章，但官方窗口当前可见的近期文章可能不足五篇。可以向下滚动官方窗口，或使用“添加文章”手动补充。

## 更新插件

如果通过 ZIP 文件安装：

1. 重新下载并解压最新版本。
2. 打开 `chrome://extensions`。
3. 找到“公众号文章助手”。
4. 点击刷新图标。
5. 回到公众号文章编辑页面，再刷新一次页面。

如果通过 Git 下载：

```bash
git pull
```

然后在 `chrome://extensions` 中刷新扩展，并刷新公众号文章编辑页。

## 本地预览

直接用浏览器打开 `preview.html`，可以在模拟编辑器中体验三个功能。模拟页面右侧包含五条公众号文章链接，用于测试“往期推荐”读取。

## 隐私与安全

- 插件不连接外部服务器。
- 文章正文和 HTML 文件只在浏览器本地处理。
- 本地设置保存在 Chrome 扩展存储中。
- HTML 导入会过滤脚本、事件属性和危险链接。
- 往期推荐只接受 `https://mp.weixin.qq.com/s/...` 格式的公众号文章链接。

## 开发与验证

安装依赖不是必需的。如果需要运行自动化检查，请确保电脑已安装 Node.js，然后执行：

```bash
npm test
npm run check
```

公众号后台改版后，优先检查：

- `src/editor-adapter.js`：正文定位、覆盖和光标插入。
- `src/content.js` 中的 `scrapeRecentArticles()`：官方超链接窗口读取。
