# 公众号文章助手

这是一个面向微信公众号官方文章编辑器的 Chrome 扩展。它提供 HTML 导入、全文关键词强调和往期推荐模板插入功能。文章内容和本地设置不会上传到外部服务。

## 安装

1. 在 Chrome 地址栏输入 `chrome://extensions`。
2. 打开右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择当前 `chajian` 文件夹。
5. 打开微信公众号后台文章编辑页，刷新一次页面。

## 使用

进入微信公众号文章编辑页后，点击页面右侧的“文章助手”按钮。也可以点击 Chrome 工具栏中的扩展图标，再点击“打开文章助手”。

### HTML 导入

1. 粘贴 HTML 代码，或选择 `.html` / `.htm` 文件。
2. 检查导入预览。
3. 选择“覆盖正文”或“插入光标位置”。
4. 点击“导入正文”。

扩展会保留正文结构、常见内联样式和图片，同时移除脚本、外部样式表、事件属性和危险链接。

### 全文关键词强调

1. 输入一个或多个关键词，每行一个，或用逗号分隔。
2. 选择黄色标记、红色加粗或蓝色下划线。
3. 点击“应用强调”。

扩展添加的强调带有内部标记。点击“移除本插件强调”只会移除扩展添加的样式，不会影响手工排版。

### 往期推荐

1. 在公众号官方编辑器中打开“超链接”窗口。
2. 在文章助手中点击“读取最近文章”。
3. 修改标题、链接或顺序；也可以删除和补充文章。
4. 从六种模板中选择一种，检查预览。
5. 选择插入正文末尾或当前光标位置。
6. 点击“插入往期推荐”。

读取官方窗口属于半自动功能。扩展会优先提取公众号文章链接；如果官方窗口只显示标题而没有暴露链接，扩展仍会读取最近五篇标题，并用黄色输入框标出需要补充的链接。补全链接后再插入正文。如果公众号后台改版导致读取失败，也可以直接在列表中手动添加文章。

## 本地预览

直接用浏览器打开 `preview.html`，可以在模拟编辑器中验证三个功能。模拟页右侧包含五条公众号文章链接，用于测试往期推荐读取。

## 维护提示

公众号后台改版后，优先检查：

- `src/editor-adapter.js`：正文定位、覆盖和光标插入。
- `src/content.js` 中的 `scrapeRecentArticles()`：官方超链接窗口读取。

## 真实后台验证清单

- 编辑页右侧可以看到“文章助手”按钮。
- HTML 文件和粘贴代码都能预览并写入正文。
- 点击正文后，可以在光标位置插入 HTML。
- 三种关键词强调可以应用，并能一键移除。
- 打开官方超链接窗口后，可以读取最多五篇文章。
- 六种往期推荐模板可以预览并插入。
- 插入后仍可在官方编辑器中继续修改和保存文章。
