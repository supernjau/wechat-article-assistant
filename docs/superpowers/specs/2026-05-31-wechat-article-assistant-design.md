# 微信公众号文章助手 Chrome 插件设计说明

## 1. 项目目标

在 `chajian` 文件夹中开发一个可通过 Chrome “加载已解压的扩展程序”安装的 Manifest V3 插件。插件面向微信公众号官方文章编辑器，为日常排版提供三个高频能力：

1. 导入 HTML 代码或本地 HTML 文件。
2. 按用户设置的关键词批量强调正文内容。
3. 从微信公众号官方“超链接”窗口读取最近发布的文章，生成并插入“往期推荐”模块。

插件不依赖第三方在线服务，不上传文章内容。首版重点是可用、稳健和方便维护。

## 2. 已确认的产品范围

### 2.1 插件入口

- 在微信公众号编辑页右侧注入一个固定悬浮按钮。
- 点击悬浮按钮后展开侧边面板。
- 同时提供 Chrome 工具栏弹窗入口。在微信公众号编辑页中可通过弹窗唤起侧边面板；不在编辑页时提示用户打开微信公众号后台。

### 2.2 HTML 导入

- 支持粘贴 HTML 代码。
- 支持选择本地 `.html` 或 `.htm` 文件。
- 支持“覆盖当前正文”和“插入到光标位置”两种模式。
- 默认使用“覆盖当前正文”。
- 导入前显示内容预览。
- 尽量保留适用于公众号文章的正文结构、内联样式和图片。
- 自动移除脚本、外部样式表、事件属性和危险链接。

### 2.3 全文关键词强调

- 支持一次输入多个关键词，可按换行或逗号分隔。
- 提供三种预设样式：
  - 黄色标记
  - 红色加粗
  - 蓝色下划线
- 插件添加的强调样式带有内部标记。
- 支持一键移除由本插件添加的强调，不影响用户手工排版。
- 只处理正文文本节点，跳过脚本、样式、代码和已生成的强调节点。

### 2.4 往期推荐

- 使用半自动流程：
  1. 用户在插件中点击“读取最近文章”。
  2. 用户打开微信公众号官方编辑器中的“超链接”窗口。
  3. 插件读取窗口中可见的最近文章，最多带回 5 篇。
  4. 用户在插入前校正文章列表。
- 列表支持修改标题、修改链接、上下移动、删除和补充。
- 支持插入到正文末尾或光标位置。
- 若官方窗口结构变化或未读取到足够内容，插件显示清晰提示，并允许用户手动补充。
- 最近一次文章列表和模板选择保存在浏览器本地。

### 2.5 往期推荐模板

首版内置 6 种使用内联样式的模板：

1. 清爽列表：细分隔线与标题列表。
2. 数字目录：使用 01-05 编号。
3. 编辑精选：每篇带小标签。
4. 极简边框：使用单侧强调线。
5. 杂志索引：标题与箭头分列。
6. 工作室风格：加入 “Super英语工作室” 品牌行。

## 3. 交互设计

侧边面板顶部显示“公众号文章助手”，主体使用三个标签页：

### 3.1 HTML 导入标签页

- 文件选择按钮
- HTML 代码输入框
- “覆盖正文 / 插入光标位置”分段选择器
- 预览区域
- “导入正文”主按钮

### 3.2 关键词强调标签页

- 多关键词输入框
- 三种样式选择
- “应用强调”主按钮
- “移除本插件强调”次按钮

### 3.3 往期推荐标签页

- 官方超链接窗口操作提示
- “读取最近文章”按钮
- 可编辑文章列表
- 新增、删除、上移和下移控制
- 6 种模板缩略预览
- “插入正文末尾 / 插入光标位置”选择
- “插入往期推荐”主按钮

面板保持紧凑，不遮挡正文主要编辑区域。窄屏环境下允许面板占用更高比例宽度。

## 4. 技术架构

插件采用 Manifest V3，主要由以下部分组成：

### 4.1 编辑器适配层

职责：

- 识别微信公众号正文编辑区域。
- 读取当前正文 HTML。
- 覆盖正文。
- 在当前光标位置插入 HTML。
- 触发编辑器需要的输入事件。
- 保存和恢复最近一次有效光标位置。

公众号后台结构变化时，优先维护此模块。

### 4.2 HTML 导入器

职责：

- 读取粘贴代码或本地文件。
- 使用 DOM 解析器提取正文。
- 清理不安全标签和属性。
- 保留可用于公众号排版的安全内联样式和图片。
- 生成预览。

过滤规则至少包括：

- 删除 `script`、`style`、`link`、`meta`、`iframe`、`object`、`embed` 和 `form`。
- 删除以 `on` 开头的事件属性。
- 删除 `javascript:` 等危险链接。

### 4.3 关键词强调器

职责：

- 解析关键词。
- 在正文文本节点中匹配关键词。
- 用带 `data-sewa-highlight` 标记的 `span` 包裹匹配内容。
- 移除本插件添加的强调节点并还原文本。

为避免短关键词抢先匹配，关键词按长度从长到短处理。

### 4.4 往期推荐生成器

职责：

- 在官方超链接窗口已打开时读取可见文章数据。
- 提取标题和链接，去重后保留最近 5 篇。
- 允许用户校正列表。
- 根据所选模板生成兼容公众号的内联样式 HTML。
- 插入正文末尾或光标位置。

读取官方窗口属于适配性功能。相关选择器集中管理，并提供手动编辑作为稳定退路。

### 4.5 本地存储

使用 `chrome.storage.local` 保存：

- 最近一次关键词输入。
- 最近一次强调样式。
- 最近一次往期推荐文章列表。
- 最近一次往期推荐模板。
- 最近一次插入位置选择。

不保存完整正文，不向外部服务传输数据。

## 5. 错误处理

- 未找到正文编辑器时，提示用户确认当前页面是否为微信公众号文章编辑页。
- 导入文件格式不正确时，不修改正文，并提示重新选择。
- 清理后的 HTML 为空时，不执行写入。
- 覆盖正文前保留当前内容快照；写入失败时恢复快照。
- 光标位置失效时，提示用户重新点击正文后再插入。
- 官方超链接窗口未打开或未读取到文章时，提示用户打开窗口，并允许手工录入。
- 读取到少于 5 篇文章时，保留已有结果并提示用户补充。

## 7. 验证方案

### 7.1 静态检查

- 校验 `manifest.json` 格式。
- 检查 JavaScript 语法。
- 确认 Manifest V3 所需权限保持最小化。

### 7.2 本地模拟验证

使用 `preview.html` 模拟公众号正文编辑器，验证：

- 粘贴 HTML 并覆盖正文。
- 选择 HTML 文件并覆盖正文。
- 在光标位置插入 HTML。
- 应用三种关键词强调样式。
- 移除插件添加的强调。
- 编辑往期推荐文章列表。
- 预览并插入 6 种模板。

### 7.3 真实后台验证

用户在 Chrome 扩展程序页面手动加载 `chajian` 文件夹后，在微信公众号官方后台验证：

- 悬浮按钮与侧边面板可以正常打开。
- 正文读取、覆盖和光标插入可用。
- 官方“超链接”窗口打开后，最近文章读取可用。
- 模板插入后可以继续在官方编辑器中修改和保存。

## 8. 首版不包含的功能

- 自动发布文章。
- 自动上传封面图。
- 云端同步模板。
- 多人协作。
- 绕过微信公众号登录或权限机制。
- 依赖公众号未公开接口批量抓取历史文章。
